## 👤 Что нужно сделать ВАМ

### 1. Запуск (каждый раз)

| Действие | Как |
|----------|-----|
| **Запустить** | Двойной клик **JARVIS** на рабочем столе |
| **Подождать** | 10–15 секунд (грузятся модели Vosk) |
| **Проверить** | Иконка в трее (стрелка `^` у часов) |
| **Остановить** | Двойной клик **JARVIS STOP** на рабочем столе |

> **Важно:** кликайте JARVIS **один раз**. Повторный клик ничего не сделает, если уже работает.  
> Если зависло — **JARVIS STOP**, подождите 5 сек, запустите снова.

### 2. Проверка после запуска (чеклист)

Отметьте у себя и сообщите результат:

```
[ ] JARVIS.lnk — запускается, ПК не зависает
[ ] Иконка в трее появляется
[ ] JARVIS STOP — останавливает
[ ] Настройки → «Прослушать приветствие» — слышен голос Дмитрия
[ ] «Джарвис» или Ctrl+J — реагирует
[ ] «Открой блокнот» / «Открой Steam» — работает
[ ] «Яркость выше» / «Громче» — работает
[ ] (опц.) Режим собеседования + Ctrl+Shift+L
```

### 3. API-ключи (уже частично настроено)

В `.env` у вас уже есть **Groq** — этого достаточно для большинства команд.

| Провайдер | Для чего | Статус |
|-----------|----------|--------|
| **Groq** | Интенты, код, собесы | 👤 Ключ в `.env` — проверить, что работает |
| **Ollama** | Код локально (опционально) | 👤 Установить [ollama.com](https://ollama.com), `ollama pull qwen2.5-coder:7b` |
| **Anthropic** | Запасной | 🔄 Trial-кредиты могли закончиться — не обязателен, если Groq работает |

Groq: [console.groq.com](https://console.groq.com) → API Keys.
